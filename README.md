RandomOrgJson-RPC
=================

A(nother) Random.org JSON-RPC API C# implementation

Features
--------

* Full implementation of the basic Random.org JSON-RPC API methods, including:
    * generateIntegers
    * generateStrings
    * generateDecimalFractions
    * generateGaussians
    * generateUUIDs
    * generateBlobs
    * getUsage

* Optional parameters
* No dependencies
